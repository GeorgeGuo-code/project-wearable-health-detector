## 2. 数据流时序（100ms 周期视角）

```
时钟:    0         10ms      20ms      30ms      40ms      50ms   ...
         │         │         │         │         │         │       │
MAX30102 │  I2C#1 read IR  │ (独立I2C)│  I2C#1 read IR  │ ...   │  100Hz
         ▼                ▼                ▼                ▼
hr_maxim    [filter+detect]      [filter+detect]      [filter+detect]
         │                │                │                │
         ▼ 7.7s 输出      ▼                ▼                ▼
hr_bpm_bin  ◄──────────── 稳态寄存器 (持续刷新) ──────────────►

MPU6050  │  I2C#2 read acc/gyro  │ (独立I2C)│  I2C#2 read ...  │ ...
         ▼                ▼                ▼                ▼
MicroBlaze   [16点MA][4态FSM]  [16点MA][4态FSM]   ...    100Hz
cadence     ◄──────── 稳态寄存器 ────────────────►
step_count  ◄──────── 累加 ──────────────────────►

DS18B20   转换 (~750ms)                                          1Hz
         ◄────────────── temperature[12] ──────────────────────►

session_stats  ┌─────── work_en 期间 ───────┐
              │  hr_sum  += heart_rate      │ (1Hz 累加)
              │  cad_sum += cadence          │
              │  hr_cnt++, cad_cnt++         │
              └──────────────────────────────┘
              ┌────── session_end ───────────┐
              │  cached_step  = step_count   │ (1 拍完成)
              │  cached_avg_hr = hr_sum/cnt  │
              │  cached_avg_cad = cad_sum/cnt│
              └──────────────────────────────┘

evaluator    每次 data_valid=1, 读 hr/temp → 算 health_status  (1Hz)

OLED cs13    持续 1MHz 刷屏, scr 由 (work_en, display_mode, flash_view_en) 决定
```

## 6. 引脚分配（建议）

```
EGO1 PMOD 排布 (示例, 最终以 XDC 为准):

PMOD JA (右):  I2C #1 - MAX30102
  JA1 ← scl1    JA2 ← 3V3    JA3 ← sda1    JA4 ← GND
  (上拉 4.7kΩ 到 3V3)

PMOD JB (左):  I2C #2 - MPU6050 + One-Wire DS18B20
  JB1 ← scl2    JB2 ← dq_temp  JB3 ← sda2    JB4 ← GND
  (sda2/scl2 上拉 4.7kΩ, dq_temp 上拉 4.7kΩ)

PMOD JC (下):  SPI OLED
  JC1 ← oled_csn    JC2 ← oled_dcn
  JC3 ← oled_clk    JC4 ← oled_dat
  (oled_rst → 单独 GPIO, 如 R3)

PMOD JD (上):  Flash SPI (预留, 本期不接)
  JD1 ← flash_csn   JD2 ← flash_sck
  JD3 ← flash_mosi  JD4 ← flash_miso
  (本期 flash_csn=1, 不连)

GPIO 杂项:
  status_led[0..2] → LD3/LD2/LD1 (绿/黄/红)
  buzzer           → 任意 GPIO, PWM 输出
  btn_mode_raw     → 拨码开关
  btn_confirm_raw  → 按键
  flash_view_en    → 拨码开关 (本期验证用)
```
